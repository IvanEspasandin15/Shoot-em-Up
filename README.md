# Shoot'em Up
Este repositorio contén un xogo para Linux inspirado no clásico Matamarcianos.

## Instruccións de Uso
Para iniciar o xogo hai que executar o archivo Build.x86_64 que se encontra na raíz do proxecto.  
O xogador terá tres vidas e controlará unha nave, coas teclas W, A, S, D e a Barra Espaciadora.  
O xogo trata de facer o maior número de puntos máximos posibles ata que se acaben as vidas do xogador.  
O xogador debe disparar coa Barra Espaciadora ás naves enemigas que se dirixen hacia el para obter puntos, cada nave destruída polo xogador daralle 25 puntos e cada nave que consiga escapar restaralle 50.  
Se a nave do xogador colisiona con unha nave enemiga ou con un asteroide, perderase unha vida.  
Se o xogador obtén 2000 puntos coa mesma vida, obterá unha extra.
